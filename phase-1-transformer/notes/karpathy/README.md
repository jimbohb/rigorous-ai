# Karpathy Source Materials

Source files from Andrej Karpathy's *Neural Networks: Zero To Hero* lecture series, used as the primary reference for Phase 1.

## Files

### `ng-video-lecture-master/`

Code and data from the **"Let's build GPT from scratch"** video lecture.

- **Source:** [karpathy/ng-video-lecture on GitHub](https://github.com/karpathy/ng-video-lecture)
- **Video:** Part of the [Zero To Hero series](https://karpathy.ai/zero-to-hero.html)
- `bigram.py` — bigram baseline language model
- `gpt.py` — full GPT implementation built up during the lecture
- `input.txt` — Tiny Shakespeare dataset used for training
- `more.txt` — extended notes from the lecture

### `../gpt_dev.ipynb` (one level up in `notes/`)

The companion Jupyter notebook to the same video — walks through the same steps interactively, from bigram model through self-attention to the full GPT, with cell outputs preserved.

- **Source:** [karpathy/ng-video-lecture on GitHub](https://github.com/karpathy/ng-video-lecture)
